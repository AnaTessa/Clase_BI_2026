# 📊 Streamlit Dashboard Base – Proyecto Final BI

Repositorio base para el desarrollo de un **tablero interactivo en Streamlit** como parte de las actividades de la materia **Especialidad en Business Intelligence** de la **Universidad Panamericana (Campus Mixcoac)**.

Este repositorio sirve como **ejemplo de estructura y flujo de trabajo** para el desarrollo del **proyecto final**, donde se integrarán procesos de análisis de datos, visualización y construcción de dashboards interactivos.

---

## 🎯 Objetivo del repositorio

Proporcionar una plantilla funcional para:

- Construir **dashboards interactivos con Streamlit**
- Integrar **procesamiento y análisis de datos**
- Aplicar conceptos de **Business Intelligence**
- Servir como **base para el proyecto final del curso**

---

## 🧠 Contexto académico

Este proyecto forma parte de la materia:

**Especialidad en Business Intelligence**  
Universidad Panamericana – Campus Mixcoac

El objetivo es que los estudiantes desarrollen un **dashboard analítico completo**, que incluya:

- Preparación de datos
- Modelado básico
- Visualización de información
- Generación de insights

---

## 🏗️ Estructura sugerida del proyecto

```
project/
│
├── app.py                # Aplicación principal de Streamlit
├── data/                 # Archivos de datos (csv, xlsx, etc.)
├── notebooks/            # Análisis exploratorio
├── src/                  # Funciones auxiliares y lógica del proyecto
│
├── requirements.txt      # Dependencias del proyecto
└── README.md             # Documentación del repositorio
```

## ⚙️ Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/usuario/repositorio.git
cd repositorio
```

2. Crear un entorno virtual (opcional pero recomendado)

```bash
python -m venv venv
source venv/bin/activate
```

3. Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## 🚀 Ejecutar la aplicación

Para correr el dashboard en Streamlit:

```bash
streamlit run app.py
```

La aplicación se abrirá automáticamente en el navegador.
